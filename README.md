# JFP.JudoFranceParis - Modernisation du Club N°1 à Paris

Ce projet GitHub est une initiative de modernisation numérique pour le club **Judo France Paris (JFP)**, le premier club de judo de la capitale avec plus de 750 adhérents et 11 dojos.

## 🎯 Objectifs du Projet

1.  **Refonte du Site Web** : Créer une interface moderne, responsive et accessible (Para-judo).
2.  **Digitalisation des Inscriptions** : Remplacer les formulaires PDF et papiers par un système de saisie en ligne.
3.  **Gestion Multi-Dojos** : Centraliser les informations sur les 11 implantations parisiennes (Paris 15, 19, 7, etc.).
4.  **Paiement Sécurisé** : Intégrer des solutions de paiement en ligne (Stripe/HelloAsso) pour faciliter le règlement des cotisations.

## 🥋 À propos de Judo France Paris

*   **Labels** : Labellisé "Or" et "Para Judo" par la FFJDA (2024-2026).
*   **Disciplines** : Judo (Baby à Senior), Jujitsu, Jujitsu Brésilien (JJB), Taïso.
*   **Rayonnement** : Présence forte dans tout Paris avec des dojos équipés.
